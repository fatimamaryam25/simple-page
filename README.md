# Cafeteria Order Management System ☕

A simple **C++ based Cafeteria Order Management System** designed to automate the process of taking orders, generating bills, and tracking daily sales in a university cafeteria.

## 📌 Problem Statement

A small university cafeteria currently handles orders manually, which causes:

* Long customer queues
* Incorrect billing
* Difficulty in tracking daily sales

This project provides a computerized solution to make the process faster, easier, and more accurate.

## 🚀 Features

* Display menu with available food items and prices
* Allow customers to place multiple orders
* Automatically calculate the total bill
* Generate an end-of-day sales summary
* Reduce manual errors in billing

## 🛠 Technologies Used

* **Language:** C++
* **Concepts:** Loops, Switch Statements, Variables, Conditional Statements

## 📂 Project Structure

* `main.cpp` → Contains the complete source code
* `README.md` → Project documentation

## 📋 Menu Items

Example menu included in the system:

1. Burger - Rs. 250
2. Pizza - Rs. 500
3. Sandwich - Rs. 200
4. Drink - Rs. 100

## ⚙️ How It Works

1. The system displays the menu.
2. The customer selects an item and enters the quantity.
3. The system calculates the cost of the selected item.
4. The customer can continue ordering more items.
5. At the end, the total bill is displayed.
6. Daily sales summary is generated.

## ▶️ How to Run
